# CSV Analyst Agent

AI-агент для автоматического анализа CSV файлов. Достаточно указать путь к файлу — агент сам разберётся в структуре данных, построит графики, найдёт закономерности и соберёт всё в красивый локальный дашборд.

Поддерживает любые CSV: продажи, логи, опросы, финансы — что угодно.

## Demo

![demo](assets/demo.gif)

## Как работает

1. Запускаешь агента и указываешь путь к CSV файлу
2. Агент читает файл, изучает структуру и планирует анализ
3. Пишет и запускает Python скрипты для построения графиков
4. Сохраняет все графики в папку `screens/`
5. Генерирует локальный HTML-дашборд с таблицами, графиками и выводами
6. В конце — краткий вывод по данным и 3 гипотезы

## Установка

```bash
pip install -r requirements.txt
```

Создай `.env` файл:

```
API_KEY=your_api_key
```

## Запуск

```bash
python main.py
```

## Возможности

- Автоматическое определение структуры данных
- Распределения, корреляции, выбросы, тренды, топы
- Сравнения по группам и категориям
- Графики сохраняются в `screens/`
- HTML-дашборд с подписями и навигацией
- Краткий вывод + 3 гипотезы по данным
- Память между сессиями — можно продолжить анализ позже
